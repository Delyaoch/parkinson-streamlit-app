# Team12-Parkinson-s-disease
Тема: Применение машинного обучения для раннего выявления болезни Паркинсона
Предмет: Раннее выявление болезни Паркинсона на основе анализа голоса с помощью машинного обучения.

Целью данной работы является проведение анализа акустических характеристик речи пациентов и построение модели машинного обучения для выявления признаков, указывающих на наличие болезни Паркинсона, и раннего выявления заболевания.
